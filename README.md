## Installation

Just clone repo and make it an executable !

```bash
git clone https://github.com/rustyinfinity/nhentai-downloader.git
```
```bash
chmod +x nhentai.sh
```

## Usage

Just enter your "Nhentai.to" URL or "Nhentai.to" ID and it will download in the current directory !
